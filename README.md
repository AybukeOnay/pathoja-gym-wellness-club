# Pathoja GYM & Wellness Club
Live Website: [pathoja.com](https://pathoja.com/)

A multi-branch fitness and wellness website built with ASP.NET Core MVC.

## Overview

Pathoja GYM & Wellness Club is a real-world web project developed to represent a multi-branch wellness brand with a responsive and user-focused interface.  
The project combines backend architecture and frontend presentation in a structured way, with dynamic content flows for services, packages, branches, and contact processes.

## Features

- Multi-branch brand structure
- Dynamic service and package presentation
- Responsive UI for desktop and mobile
- Branch-based content organization
- Contact and lead form flows
- Map, QR, and branch information integration
- Layered architecture for maintainable development
- Backend-focused project structure with clear separation of responsibilities

## Technologies Used

### Backend
- ASP.NET Core MVC
- C#
- Entity Framework Core
- PostgreSQL

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

### Architecture & Patterns
- Layered Architecture
- DTO / ViewModel structure
- AutoMapper
